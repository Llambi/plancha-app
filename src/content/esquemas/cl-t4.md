- Comunicar para crear valor
  - Finalidad
    - Actividad interactiva inherente a la persona
    - Busca <b>influir en el comportamiento</b> del otro
    - Genera <b>redes y sinergias</b>, no solo retroalimentación
    - Las competencias comunicativas <b>se aprenden</b>
- Comunicación y percepción
  - Acto perceptivo
    - Registro de datos + interpretación automática de los sentidos
    - Clarificar el mensaje para una lectura <b>neutral y ecuánime</b>
- El proceso comunicativo
  - Elementos
    - <b>Emisor</b>: origen del mensaje
    - <b>Receptor</b>: descifra mediante el código
    - <b>Mensaje</b>: información codificada
    - <b>Canal</b>: medio por el que se materializa
    - <b>Código</b>: sistema compartido para cifrar/descifrar
  - Cuándo se completa
    - Solo cuando el receptor entiende con la <b>intención del emisor</b>
- Es imposible no comunicar
  - Axioma
    - Todo comportamiento comunica, también el silencio
    - Coherencia entre el canal verbal y el no verbal (gestual)
