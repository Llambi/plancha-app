- Qué es
  - Información comprensible por máquinas; RDF es a la web semántica lo que HTML a la web usual
  - Tecnologías base: <b>RDF, RDFS, OWL, SPARQL</b>
- Datos enlazados (Linked Data) — Berners-Lee
  - 1) URIs para identificar recursos
  - 2) HTTP URIs para acceder a ellos
  - 3) Ofrecer información con RDF/SPARQL
  - 4) Enlazar las URIs entre sí
- Sistema de 5 estrellas
  - Usar estándares web (HTTP, RDF, SPARQL) y enlazar con datos de otros proveedores
- Aplicaciones
  - Buscadores semánticos, soporte a la decisión, gestión documental, apps empresariales
  - NO: páginas web estáticas
- Minería y Web 3.0
  - Minería ontológica y de la web semántica
  - <b>Web 3.0</b>: descentralizada, con confianza, privacidad y seguridad
