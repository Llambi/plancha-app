titulo: Evaluar un dataset con las 5 estrellas
kind: practico
enunciado: Un ayuntamiento publica sus presupuestos. Indica qué necesitaría para subir desde 1 hasta 5 estrellas en el esquema de datos abiertos enlazados de Berners-Lee.
orden: 107
---

<ul>
<li>★ Publicar los datos en la web con una licencia abierta (aunque sea un PDF).</li>
<li>★★ Publicarlos como datos estructurados procesables (p. ej. una hoja de cálculo en vez de PDF).</li>
<li>★★★ Usar un formato <b>no propietario</b> (p. ej. CSV en lugar de Excel).</li>
<li>★★★★ Usar estándares de la W3C (<b>URIs y RDF</b>) para identificar las cosas, de modo que se pueda enlazar.</li>
<li>★★★★★ <b>Enlazar</b> los datos con datos de otros proveedores para dar contexto (Linked Open Data).</li>
</ul>
