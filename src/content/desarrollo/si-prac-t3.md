<ul>
<li>(1) Tabla SQL → <b>estructurado</b> (organización, posición y tamaño definidos).</li>
<li>(2) JSON → <b>semiestructurado</b>. (4) XML → <b>semiestructurado</b>. (3) Texto del correo → <b>no estructurado</b>.</li>
<li><b>Dato</b>: «38». <b>Información</b>: «la temperatura del paciente es 38 ºC» (dato + significado).</li>
<li><b>Conocimiento</b>: «38 ºC indica fiebre, conviene vigilar» (información asimilada y contextualizada por el sujeto).</li>
</ul>
