<ul>
<li>1) Usar <b>URIs</b> para identificar los recursos.</li>
<li>2) Usar <b>HTTP URIs</b> para poder acceder a dichos recursos.</li>
<li>3) Al acceder, ofrecer información con estándares como <b>RDF o SPARQL</b>.</li>
<li>4) Fijar <b>enlaces</b> entre las URIs de los recursos para facilitar el descubrimiento de información.</li>
<li>Sobre estos principios se define el sistema de 5 estrellas de apertura/usabilidad.</li>
</ul>
